Bulk Editing Media

There are a couple of tools that can be used to bulk edit media within Jinzora. These can be accessed from the "Media Management" menu when logged in with admin privledeges (most of these are accessable on the "Tracks" page). They are:

Retag All Tracks
This tells Jinzora to dynamically rewrite the ID3 tags on the tracks based on the filesystem data. It will also, if present, write the album art directly to the ID3 tag (so when the track is streamed the album art can be viewed).

When writing the tags the filesystem data is used. For example in "Genre" mode (or 3 directory mode) the following file would be written the following way:

/Jazz/Miles Davis/Kind of Blue/07 - All Blues.mp3

Genre: Jazz
Artist: Miles Davis
Album: Kind of Blue
Track Name: All Blues
Track Number: 07
Renumber Tracks
This will tell Jinzora to use it's meta data retrival functions to attempt to renumber tracks for an album.



Search & Replace
This will also you to edit the names of tracks and bulk remove information. Say each track has the artist name in it, and you'd like to clean it up by removing that.

Fix Filename Case - This will change a track name from:

07 - all blues.mp3

to

07 - All Blues.mp3

So it will display prettier.



Fix Filename Case
This will change a track name from:

07 - all blues.mp3

to

07 - All Blues.mp3

So it will display prettier.



Create Clips | Delete Clips
This allows you to bulk create "clip" tracks from all the tracks in an album. This will create a songname.clip.mp3 file using approximately 30 seconds of data starting 30 seconds into the track.



Create LoFi Tracks | Delete LoFi Tracks
This allows you to create pre-resampled tracks (using lame) for streaming from Jinzora. Using this tool you can pre-create these tracks to they do not have to be resampled on the fly.
