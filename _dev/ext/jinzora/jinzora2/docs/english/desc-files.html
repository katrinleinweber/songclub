Description Files

There are many places within Jinzora that you can specify descriptions of items. To add descriptions for items use the "Media Management" - "Item Information" tool. Be sure to select this tool when viewing the item you wish to edit.

Genre/Artist At the Genre/Artist level Jinzora can display a description of each just below the name of the Genre/Artist.



Genre Sub Description If you wish, when viewing a Genre (meaning you've clicked on the link for the Genre and are now looking at all the artists in that Genre, or the Artists page) you can add an image and description file to the top area.



Album View At the Album View you can add a description file describing a particular artist.



Track View:
Album Description 		At the track view you can also include a description of the album that you are currently viewing.



Track View:
Track Description 		At the track view you can also include a description of the individual tracks. This is done by editing the actual track using the edit dialog box (the ? in most frontends).
