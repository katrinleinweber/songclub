Upgrading Jinzora on Windows Systems

Currently there is no upgrade path from Jinzora 1.1 to Jinzora 2.0. Due to the nature of Jinzora 2.0's powerful backend systems you must fully install Jinzora 2.0 and manually move any user accounts to Jinzora 2.0. We will support upgrades from Jinzora 2.0 to future versions. when you are ready to begin the installation process follow one of our guides below

  • Standalone Install

  • Postnuke Install

  • PHPNuke Install
