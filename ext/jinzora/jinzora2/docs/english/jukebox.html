Jukebox Install

You can use Jinzora's Jukebox mode to play media on the Jinzora server rather than stream it to another computer. This allows you to put a computer with your home stereo and control meida playback to it using Jinzora.

The currently supported Jukebox interfaces are:
  • MPD (Music Player Daemon) on *NIX platforms
  • Winamp (with the httpQ plugin) on Windows platforms
  • SlimServer by SlimDevices - a hardware device
  • The Turtle Beach Audiotron - a hardware device

    • It is assumed that the jukebox interface you are planning on using is already working before proceeding. Please ensure that either MPD, Winamp (with httpQ), SlimServer, or your Audiotron can play media just fine before using the guide below.

    Setting up Jukebox Mode

  • Make sure Jinzora is up and running fully before beginning

  • Edit the Jinzora settings.php file and set $jukebox = "true";

  • Refresh your Jinzora interface. You should now have a "Jukebox bar". You'll now need to edit settings.php in the Jinzora "jukebox" directory (not the root one).
