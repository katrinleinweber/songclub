Random Album Art

Jinzora can display random album art from a random sampling of albums from your collection. This can be displayed on the home page as well as the artist pages. There are a few settings for random album display, they are:
  • $random_albums
    This is the number of "slots" to put in the Random Album Section


  • $random_per_slot
    How many different albums should we cycle between in each "slot"


  • $random_rate
    How quickly should we rotate the images (in ms, so 1000 = 1 second)


  • $random_art_size
    The dimensions ("100x100") that the art should be display at.
