Using the Embedded Media Players

Jinzora supports a very powerful embedded player system. What this means is that when the user selects media to play it does not stream to a local client, but rather opens a popup window with an embedded player to play the music. The goal of this system is to make it much more difficult for users to gain access to your media without your permission while still allowing them to play media. There are currently 4 supported players supporting different media and options. They are:

  • xspf (XSPF Player)
    Benefits
    * Simple but elegant interface
    * Flash Based for great compatibility
    * Open source and free!


  • wimpy (Wimpy Flash Based Player)
    Benefits
    * Wonderful interface (album art, playlist, etc)
    * Flash Based for great compatibility
    Drawbacks
    * NOT Free ($50 cost)


  • wmp (Windows Media Player)
    Benefits
    * Supports both Audio and Video
    * Works very well with Windows system
    * Plays a wide variety of file formats
    Drawbacks
    * Windows only (although depending on the config could work in Linux/Mac)
    * No graphical playlist while playing multiple files
