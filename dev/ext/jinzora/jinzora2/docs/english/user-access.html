User Manager

Jinzora can be used in a multi-user environemt, with different users having different rights. You create and edit users in the "User Manager" in the Jinzora "System Tools" section. Using this tool you can:
  • Add a user
  • Edit default system access
  • Modify an existing user
  • Edit self-registration settings
  • Remove users


    • All options in the user manager are well documented within the user manager.
